## Purpose / 目的

License information exchange Subgroup discusses the guideline for exchanging license info. between organizations and SPDX Lite.

ライセンス情報サブグループでは、組織間のライセンス情報授受に関してのガイドラインやSPDX Liteといった仕様を検討しています。
